## 🚀 Como executar

### Pré-requisitos

* [Python 3.x](https://www.python.org/downloads/) instalado

### Executando os exercícios

```bash
# Clone o repositório
git clone https://github.com/Gustavolg95/python_revisao_logica.git

# Acesse a pasta
cd python_revisao_logica

# Execute um exercício
python ques_01.py
```
